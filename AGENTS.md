# Agent 读取说明

这个仓库是用户的第二大脑知识库，主要服务 Codex 或其他 coding agent。

新任务开始时，不要扫描整个仓库。先读：

- `90_索引/Agent 启动上下文.md`
- `10_知识库/偏好/AI 协作偏好总览.md`
- `90_索引/问题到方法群路由.md`

核心原则：

- 所有知识都服务于下一次相似任务。
- 不要把一次性现场事实当长期知识。
- 知识卡片默认中文、无代码、偏方法论。
- 工作对话里不要强行复盘。
- 新材料默认先进入 `00_收件箱/夜间复盘候选.md`，晚上复盘后再决定是否写入长期知识层。

写回知识库前先读：

- `_系统/隐式材料收件箱与夜间微信复盘规则.md`
- `_系统/新内容导入 SOP.md`
- `_系统/Codex 对话沉淀 SOP.md`
- `_系统/收件箱处理规则.md`

如果当前任务来自服务器或远程 agent：

- 优先读取相对路径，不要依赖本机绝对路径。
- 如果需要同步最新知识库，先执行 git pull。
- 如果在服务端产生了新复盘或方法修正，写入后通过 git 提交并推送，避免只存在远程临时上下文里。
