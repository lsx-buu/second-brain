# 无 Obsidian 使用说明

你不需要安装 Obsidian 也能使用这个知识库。

## 最简单入口

双击打开：

打开知识库.html

然后在页面左侧点击“选择知识库文件夹”，选择当前这个文件夹：

我的第二大脑

浏览器会读取本地 Markdown 文件，并提供搜索和阅读。

## 日常怎么用

1. 新的文字、文档摘要、聊天记录，先放到 00_收件箱。
2. 让 Codex 先判断哪些内容离开当前项目仍然有用。
3. 只把可迁移判断提炼成无代码知识卡片。
4. 优先从 90_索引/首页 的问题入口判断它该补哪张母卡。
5. 再打开“打开知识库.html”，重新选择文件夹即可看到新内容。

## 可以用哪些软件编辑

任选一个即可：

- Windows 记事本
- VS Code
- Typora
- Notepad++
- 任意 Markdown 编辑器

## 以后再升级

当卡片超过 20 到 50 张后，可以再接：

- Khoj：更偏个人第二大脑问答。
- Open WebUI：更偏本地 AI 聊天和知识库。
- Codex Hooks：自动把重要对话提炼成草稿。

前期重点不是工具，而是稳定沉淀。
